###### FastCampus
## Front-End Design CAMP

`웹 타이포그래피`/`그리드시스템`/`버티컬리듬`/`황금비율`을 기반으로 하는 모던 웹 디자인 방법론을 배워봅니다. 디자인 방법론을 도입한 웹 시안을 디자인한 후, 웹 데이터(HTML/CSS)로 제작해보는 과정을 공부하게 됩니다. 디자인부터 제작까지 흥미로운 여정이 되겠지요!
